# CODE-A-NOVA 🚀

A collection of Python projects developed during my internship and learning journey. This repository showcases practical applications of Python programming, problem-solving, automation, and software development fundamentals.

## 📌 Projects Included

### 1️⃣ Password Generator

A command-line application that generates secure and customizable passwords.

#### Features

* Generate strong random passwords
* User-defined password length
* Supports letters, numbers, and special characters
* Fast and lightweight

#### Concepts Used

* Python Functions
* Random Module
* String Manipulation
* User Input Handling

---

### 2️⃣ To-Do List Application

A command-line task management system that helps users organize and track their daily tasks.

#### Features

* Add Tasks
* View Tasks
* Delete Tasks
* Persistent Task Storage
* Simple and User-Friendly Interface

#### Concepts Used

* File Handling
* Lists
* Loops
* Conditional Statements
* CRUD Operations

---

### 3️⃣ Email Sender CLI

A Python-based email automation tool that sends emails using Gmail SMTP.

#### Features

* Send Emails from Command Line
* Custom Receiver Email
* Custom Subject
* Custom Message Body
* SMTP Authentication
* Error Handling

#### Concepts Used

* SMTP Protocol
* EmailMessage Class
* Exception Handling
* User Input Processing
* Network Communication
