# 🔐 Password Generator

A simple Python-based Password Generator that creates secure and customizable passwords using a combination of letters, numbers, and special characters.

## 📌 Description

This project allows users to generate strong passwords by choosing:

* Password length (minimum 8 characters)
* Whether to include numbers
* Whether to include special characters

The program ensures that the generated password satisfies the selected requirements and helps users create safer passwords for their accounts.

## ✨ Features

* Generate passwords of custom length
* Minimum password length validation
* Option to include numbers
* Option to include special characters
* Uses random character selection for better security
* Simple command-line interface

## 🛠️ Technologies Used

* Python 3
* `random` module
* `string` module

## 🚀 How to Run

1. Clone the repository:

```bash
git clone <your-repository-link>
```

2. Navigate to the project folder:

```bash
cd Password-Generator
```

3. Run the program:

```bash
python passwordGenerator.py
```
